# UTM-конвенции Evolution House

Используйте UTM только во внешних ссылках на сайт; не добавляйте их к внутренней навигации.

| Параметр | Формат | Пример |
|---|---|---|
| `utm_source` | `instagram`, `tiktok`, `facebook`, `youtube`, `telegram`, `email` | `instagram` |
| `utm_medium` | канал распространения | `organic_social` |
| `utm_campaign` | нижний регистр, слова через `_` | `reels_july_2026` |
| `utm_content` | идентификатор конкретного материала | `reel_03` |
| `utm_term` | только для согласованной кампании | `archetypes` |

Пример: `?utm_source=instagram&utm_medium=organic_social&utm_campaign=reels_july_2026&utm_content=reel_03`.
