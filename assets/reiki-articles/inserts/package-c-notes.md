## QA

- Проверены viewport `1440px`, целевой контейнер `760px`, а также mobile `360px` и контрольный `340px`.
- На проверенных ширинах горизонтальный scroll отсутствует.
- Текст остаётся живым HTML и читается скринридером; декоративные SVG скрыты через `aria-hidden`.
- У каждого `figure` есть связанный заголовок и развёрнутое скрытое описание.
- Композиции на mobile перестраиваются, а не масштабируются целиком.
- Для тёмного фрагмента 16 зафиксирован явный светлый цвет заголовка; для светлого фрагмента `18/four-situations-matrix.html` зафиксированы явные тёмные цвета заголовков и текста, чтобы стили страницы-хоста не снижали контраст.
- Исходные статьи, генератор, общие CSS и sitemap в рамках пакета не изменялись.
